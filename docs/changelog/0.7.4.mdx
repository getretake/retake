## pg_search

### Bug Fixes

- Fixed an issue where indexing woudl be incorrect following a `DROP COLUMN` operation.

## pg_lakehouse

### New Features

- Added support for running `EXPLAIN` and `EXPLAIN ANALYZE` queries over foreign tables.

### Bug Fixes

- Fixed a bug where users could not connect to Azure Blob Storage and ADLS Gen2.
- Fixed a bug where the Tokio reactor would crash following an incorrect query.
