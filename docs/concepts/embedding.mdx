---
title: "Embedding"
---

`Embedding` specifies the embedding model we wish to use. Retake provides lightweight wrappers for
most popular embedding models and allows you to provide your own model.

## Sentence Transformers

**Usage**

```bash
embedding = Embedding.SentenceTransformer(model="all-MiniLM-L6-v2")
```

**Parameters**

<ParamField body="model" type="string" default="all-MiniLM-L6-v2">
  SentenceTransformer model name
</ParamField>

## OpenAI

**Usage**

```bash
embedding = Embedding.SentenceTransformer(
    api_key="***",
    model="text-embedding-ada-002"
)
```

**Parameters**

<ParamField body="api_key" type="string" required>
  Your OpenAI API key
</ParamField>
<ParamField body="model" type="string" required>
  OpenAI model name
</ParamField>

## Custom Model

**Usage**

```bash
from typing import List

def custom_model (docs: List[str]) -> List[List[float]]:
  # Custom logic here
  return [[0.1, 0.2, 0.3], [0.1, 0.2, 0.4]]

embedding = Embedding.Custom(
    func=custom_model
)
```

**Parameters**

<ParamField body="func" type="function" required>
  Custom embedding function. Must accept a string array and return an array of
  float arrays.
</ParamField>
