---
title: "Sink"
---

`Sink` specifies where we want to store our vectors. Retake provides lightweight wrappers around
all supported sinks.

## Pinecone

**Usage**

```bash
sink = Sink.Pinecone(
    api_key="***",
    environment="asia-southeast1-gcp"
)
```

**Parameters**

<ParamField body="api_key" type="string" required>
  Pinecone API key
</ParamField>
<ParamField body="environment" type="string" required>
  Pinecone environment
</ParamField>

## Weaviate

**Usage**

```bash
sink = Sink.Weaviate(

    url="weaviate_url"
)
```

**Parameters**

<ParamField body="api_key" type="string" required>
  Weaviate API key
</ParamField>
<ParamField body="url" type="string" required>
  Weaviate URL
</ParamField>

## ElasticSearch

**Usage**

```bash
sink = Sink.ElasticSearch(
    cloud_id="cloud_id"
)
```

**Parameters**

<ParamField body="cloud_id" type="string">
  ElasticSearch cloud ID, if using the cloud version
</ParamField>
<ParamField body="host" type="string">
  ElasticSearch host address, if running locally
</ParamField>
<ParamField body="user" type="string">
  ElasticSearch username
</ParamField>
<ParamField body="password" type="string">
  ElasticSearch password
</ParamField>
<ParamField body="ssl_assert_fingerprint" type="string">
  ElasticSearch HTTP CA certificate SHA-256 fingerprint, if running locally
</ParamField>
