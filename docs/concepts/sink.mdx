title: "Sink"
---

`Sink` specifies where we want to store our vectors. Retake provides lightweight wrappers around
all supported sinks.

## ElasticSearch

**Usage**

```bash
sink = Sink.ElasticSearch(
    cloud_id="cloud_id"
)
```

**Parameters**

<ParamField body="cloud_id" type="string">
  Your ElasticSearch cloud ID, if using the cloud version
</ParamField>
<ParamField body="host" type="string">
  ElasticSearch host address, if running locally
</ParamField>
<ParamField body="user" type="string">
  ElasticSearch username
</ParamField>
<ParamField body="password" type="string">
  ElasticSearch password
</ParamField>
<ParamField body="ssl_assert_fingerprint" type="string">
  ElasticSearch HTTP CA certificate SHA-256 fingerprint, if running locally
</ParamField>
